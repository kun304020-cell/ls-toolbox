# LSL

LSL 是一个面向个人 AI 工作与陪伴体验的 Windows 桌面应用。

它把 LSL 原生主脑、外部 Agent、长期记忆、任务编排与角色陪伴统一到同一个工作空间中，同时保持不同角色、房间和工作上下文之间的边界。

## 当前能力

- LSL 原生主脑对话与长期记忆
- MiMoCode、OpenClaw、Codex CLI 等 Agent 接入
- 多 Agent 任务编排与统一结果汇总
- 结构化消息渲染，包括标题、代码、差异、工具过程、媒体与错误状态
- 统一的发送、停止、修改、撤回、恢复、搜索与导出操作
- 角色卡、陪伴房间、Director 场景组织与多角色对话
- 角色长期记忆和角色房间记忆硬隔离
- 陪伴对话实时流式输出与中途停止

## 下载

请从 [Releases](https://github.com/kun304020-cell/ls-toolbox/releases) 下载最新 Windows 安装包。

推荐普通用户下载 `LSL_1.0.0_x64-setup.exe`。发布附件同时提供 MSI 和 SHA-256 校验文件。

## 测试版说明

- 当前版本是小范围预览测试版。
- 安装包尚未进行代码签名，Windows 可能显示安全提醒。
- API 密钥、会话和本地记忆不会包含在安装包或本仓库中。
- 外部 Agent 功能需要用户自行安装并配置相应运行环境。
- 测试重要或破坏性功能前，请先备份本地会话数据。

## 反馈

如遇到安装、Agent 连接、流式输出、记忆隔离或消息渲染问题，请在 [Issues](https://github.com/kun304020-cell/ls-toolbox/issues) 中提交说明和可公开的截图。请勿上传 API 密钥、私人对话或其他敏感信息。
